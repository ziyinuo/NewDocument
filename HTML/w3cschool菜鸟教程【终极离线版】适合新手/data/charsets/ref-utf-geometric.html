UTF-8 几何图形（Geometric Shapes）

范围：十进制 9632-9727，十六进制 258A0-25FF。

如果您想要在 HTML 中显示这些字符，您可以使用下表中的 HTML 实体。

如果字符没有 HTML 实体，您可以使用十进制或十六进制引用。

实例

<p>我将显示 &#9658;<p>
<p>我将显示 &#x25BA;<p>
