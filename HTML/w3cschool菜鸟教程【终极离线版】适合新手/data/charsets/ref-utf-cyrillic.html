UTF-8 Cyrillic（西里尔字母）

范围：十进制 1024-1279，十六进制 0400-04FF。

如果您想要在 HTML 中显示这些字符，您可以使用下表中的 HTML 实体。

如果字符没有 HTML 实体，您可以使用十进制或十六进制引用。

实例

<p>我将显示 &#1025;<p>
<p>我将显示 &#x0401;<p>

结果如下：

我将显示 Ё
